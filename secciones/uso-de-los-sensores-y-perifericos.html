IMPORTANTE

Para usar los sensores de su Datapek, asegúrese de:

  • Tener encendido su Datapek.
  • Tener conexión por VNC y/o SSH
  • Haber clonado el repositorio oficial de Datapek
  • Estar dentro del repositorio que clonó de Datapek
  • Ejecutar el comando 
    chmod +x instalar_dependencias.sh
  • Ejecutar el comando 
    ./instalar_dependencias.sh

Con estos pasos su Datapek está listo para usarse.

¿Cómo uso mi Datapek?

Para el uso de todos los sensores, desde su Datapek, asegúrese de estar en el directorio que clonó de Datapek oficial en la sección de Bienvenida/Manual de Usuario - Datapek.

Para grabar ejecute el comando 

./ejecutar.sh
se desplegará un menú como el siguiente:

Siéntase libre de elegir el número de las opciones que usted desée dentro del menú de opciones, una vez que esté listo seleccione la opción número 5 para ejecutar el programa.

¿Cómo recupero los datos y videos recolectados?

Una vez que ha ejecutado el programa y su Datapek ha terminado la recolección de datos, querrá poder visualizarlos, para ello, partiendo del repositorio oficial clonado de Datapek, utilice el comando 

cd nombre_del_perro/nombre_de_la_prueba
para moverse a la carpeta donde se encuentran sus resultados.

Donde:

  • nombre_del_perro se refiere al nombre que usted le asignó durante la ejecución del programa al perro, en el ejemplo usamos el nombre de solovino
  • nombre_de_la_prueba se refiere al nombre que usó para la prueba del perro, en el ejemplo usamos el nombre de prueba-1
