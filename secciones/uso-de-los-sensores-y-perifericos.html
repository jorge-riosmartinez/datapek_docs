Prueba de diagnóstico de los Sensores y Periféricos

Para probar el correcto funcionamiento de tu Datapek primero asegúrate de haber establecido conexión por SSH y estar dentro del repositorio que clonaste anteriormente en la sección Configuración del Software, esto lo puedes hacer primero ubicando tu directorio/carpeta actual de trabajo con el comando:

pwd

De esta forma sabrás el directorio en el que te encuentras, si no estás dentro del repositorio clonado de Datapek, entonces muestra los directorios que se encuentran dentro de tu directorio actual con el comando:

ls -la

Prueba a moverte un directorio que haya resultado del comando anterior con el comando:

cd nombre_de_la_carpeta

Repite estos pasos hasta estar dentro del repositorio de Datapek. En caso de que quieras retroceder una carpeta puedes usar el comando:

cd ..

Una vez que estás seguro de estar dentro del repositorio que clonaste de Datapek, asegurate de comprobar el correcto funcionamiento de los sensores, ejecutando el script de diagnóstico:

python firmware/utils/diagnostico.py

Si todos los sensores funcionan correctamente entonces estás listo para iniciar la recolección de datos, para ello usa el script adquisicion_datos.py con los siguientes argumentos:

python firmware/adquisicion_datos.py -n firulais -p prueba-1 -d 10 --imu --camara --audio --temperatura

Descripción de los parámetros:

  • -n, --nombre_perro: Nombre del perro. Por ejemplo, firulais.
  • -p, --prueba: Nombre de la prueba. Por ejemplo, prueba-1.
  • -d, --duracion: Duración de la prueba en segundos. Por ejemplo, 10.
  • --imu: Activa la recolección de datos del IMU (Unidad de Medición Inercial).
  • --camara: Activa la grabación de video de la cámara.
  • --audio: Activa la grabación de audio del micrófono.
  • --temperatura: Activa la recolección de datos del sensor de temperatura ds18b20.

Ejemplo de uso:

python firmware/adquisicion_datos.py -n firulais -p prueba-1 -d 10 --imu --camara --audio --temperatura

Este comando recopilará datos durante 10 segundos del perro llamado firulais para la prueba llamada prueba-1. Los datos incluirán lecturas del IMU, grabación de video, grabación de audio y datos de temperatura.
