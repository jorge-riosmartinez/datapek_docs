Solución de Problemas

  • Problema: No puedo ver una sesión con VNC.

    • Use el comando 

    sudo raspi-config
    para acceder a las configuraciones de su Raspberry Pi.

    Navegue a System Options -> Boot / Auto Login -> Desktop / CLI y seleccione la opción B4 Desktop GUI, automatically logged in as 'datapek' user.
