Instalación de Bibliotecas y Dependencias

Comandos para instalar las bibliotecas necesarias:

sudo apt-get update
sudo apt-get install python3
pip install RPi.GPIO

Configuración del Software

  1. Prepara la tarjeta SD con el sistema operativo Raspberry Pi OS.
  2. Configura el usuario y la contraseña durante la instalación inicial.
  3. Habilita SSH y encuentra la dirección IP de la Raspberry Pi.
  4. Conéctate a la Raspberry Pi mediante SSH.
  5. Ejecuta los comandos para instalar las dependencias necesarias.

Instalando dependencias

Para instalar todas las dependencias necesarias ejecuta el script instalar_dependencias.sh de la siguiente forma:

chmod +x instalar_dependencias.sh
