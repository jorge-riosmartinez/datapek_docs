Configuración del Software

Instalación del Sistema Operativo

Para comenzar, debes preparar una tarjeta SD con el sistema operativo Raspberry Pi OS. Puedes descargar el Raspberry Pi Imager desde el sitio oficial.

Configuración Inicial

Una vez que hayas instalado el sistema operativo, configura el usuario y la contraseña durante la instalación inicial. Habilita SSH para acceder a tu Raspberry Pi de forma remota y encuentra la dirección IP de la Raspberry Pi en tu red local.

Instalación de Bibliotecas y Dependencias

Conéctate a la Raspberry Pi mediante SSH y ejecuta los siguientes comandos para instalar las bibliotecas necesarias:

sudo apt update && sudo apt upgrade

Instalando dependencias

Para simplificar la instalación de todas las dependencias necesarias, hemos creado un script llamado instalar_dependencias.sh. Para ejecutarlo, sigue estos pasos:

  1. Descarga el script desde el repositorio de Datapek:
    2. git clone https://github.com/jorge-riosmartinez/datapek_docs.git
  2. Navega al directorio donde se encuentra el script:
    3. cd datapek_docs
  3. Haz que el script sea ejecutable:
    4. chmod +x instalar_dependencias.sh
  4. Ejecuta el script:
    5. ./instalar_dependencias.sh

Este script se encargará de instalar todas las dependencias y bibliotecas necesarias para que tu Datapek funcione correctamente.
