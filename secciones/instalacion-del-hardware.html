Instrucciones Paso a Paso

  1. Conexión del giroscopio...

    Descripción: El módulo MPU-9250 es un IMU de 9 grados de libertad que integra un acelerómetro, un giroscopio y un magnetómetro de 3 ejes, además de un co-procesador DMP para aliviar la carga del microcontrolador principal. Es ideal para proyectos que requieren estas mediciones y se conecta al microcontrolador vía I2C.

  2. Conexión de la cámara...

    Descripción: El Raspberry Pi Camera Model 2 cuenta con un sensor Sony IMX219 de 8 megapíxeles, mejorando la calidad de imagen, fidelidad de color y rendimiento en baja luz respecto al sensor de 5 megapíxeles del modelo anterior. Puede capturar video en 1080p30, 720p60 y VGA90, además de fotografías. Compatible con todos los modelos de Raspberry Pi 1, 2, 3 y 4, se conecta a través de un cable de 15 cm al puerto CSI y es accesible mediante las APIs MMAL y V4L, así como la biblioteca Picamera.

  3. Conexión del sensor de temperatura...

    4. Descripción: Sensor de temperatura casero mini-jack al sensor.

    Para poder usar el sensor de temperatura se debe conectar el cable mini-jack que viene separada del Datapek. Este es el único caso donde debemos hacer esto.

  4. Conexión del micrófono...

    Descripción: Micrófono USB genérico
